# デモ配布セットアップ

## 必要なもの

- Docker
- Docker Compose
- Anthropic APIキー

APIキーが未設定でも画面操作はできますが、AIリアクションは固定文にフォールバックします。

## 配布ファイルを作る側

開発マシンで以下を実行します。

```bash
./scripts/make-demo-package.sh
```

生成先:

```text
dist/nyanta-chat-demo-YYYYMMDD-HHMM.tar.gz
```

## 受け取った側の起動手順

サーバへ圧縮ファイルを置き、展開します。

```bash
tar xzf nyanta-chat-demo-*.tar.gz
cd nyanta-chat
```

環境変数ファイルを作ります。

```bash
cp .env.example .env
```

`.env` を編集します。

```env
ANTHROPIC_API_KEY=your_anthropic_api_key_here
DB_PATH=./data/nyanta.db
```

起動します。

```bash
mkdir -p data
docker compose up -d --build
```

ブラウザで開きます。

```text
http://サーバのIPまたはドメイン:3001/nyanta
```

## 停止

```bash
docker compose down
```

## ログ確認

```bash
docker compose logs -f
```

## データ保存場所

回答データはSQLiteで保存されます。

```text
./data/nyanta.db
```

`data/` フォルダを削除すると、保存済み回答も消えます。
