# Notionインポート手順

このフォルダ内のMarkdownファイルをNotionへインポートできます。

## 手順

1. Notionを開く
2. 左サイドバーの `Import` を選ぶ
3. `Markdown & CSV` を選ぶ
4. `notion-export/` 内の `.md` ファイルを選んでインポート

おすすめ順:

1. `00_にゃんこ先生チャット_概要.md`
2. `01_デモ配布セットアップ.md`
3. `02_開発運用メモ.md`
4. `03_変更履歴.md`
5. `04_今後の機能案.md`

Notion上で1つの親ページを作り、その中に各Markdownを子ページとして入れると整理しやすいです。
