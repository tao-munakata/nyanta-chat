# にゃんこ先生チャット 概要

## これは何か

にゃんこ先生チャットは、訪問診療・在宅医療の初回相談に向けて、利用者がチャット形式で問診項目を入力できるデモアプリです。

猫の医師キャラクター「にゃんこ先生」が、やさしい口調で質問しながら情報を集めます。最後に回答まとめを表示し、保存できます。

## 想定用途

- 訪問診療前の事前問診
- 家族や介護者による代理入力
- デモ・プロトタイプ検証
- クリニック向けUI案の説明

## 主な機能

- 問診質問25問、7カテゴリ
- テキスト、日付、選択肢、写真入力
- 1つ前の質問へ戻る
- 残り質問数の表示
- 回答まとめ画面
- 保存ボタン
- 電話番号チェック
- 生年月日チェック
- 質問カテゴリごとの猫キャラクター切り替え
- 猫キャラクターのまばたき・口パクアニメーション
- SQLite保存
- Docker起動

## 技術構成

| 項目 | 内容 |
| --- | --- |
| フレームワーク | Next.js App Router |
| UI | React / Tailwind CSS |
| DB | SQLite / better-sqlite3 |
| AIリアクション | Anthropic Claude API |
| 配布 | Docker Compose / tar.gz |
| 公開パス | `/nyanko` |
