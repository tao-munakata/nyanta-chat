# 変更履歴

## 未リリース

なし

## v0.8

- `nyanko-medical`、`nyanko-smalltalk`、`nyanko-mood`、`nyanko-secret` の4サービス構成を追加
- `/nyanko` をサービス一覧に変更し、既存問診票を `/nyanko/medical` に移動
- 雑談、気持ち整理、匿名相談の自由チャット画面を追加
- テーマごとに猫キャラクター、色、注意文、システムプロンプトを切り替え
- Anthropic APIキー未設定時でも、入力内容に応じて返信が変わる簡易返信を追加

## v0.6

- 完了画面っぽい見た目の「質問編集ページ」を追加
- 質問文をカテゴリごとに編集して保存できるように変更
- 編集した質問文が、問診画面・完了画面・CSV出力に反映されるように対応
- 問診トップから質問編集ページへ移動できる導線を追加

## v0.45

- 完了画面にCSV保存ボタンを追加
- 入力途中で閉じても同じ端末で続きから再開できるように変更
- 戻るボタンで修正した回答が再開位置に反映されるように保存処理を調整
- 重要ポイントまとめを「要確認」「注意」「参考」に分類して表示
- CSV確認ページでCSVファイル保存とCSVコピーの両方に対応

## v0.4

- 完了画面に「重要ポイントまとめ」を追加
- 痛み、主な症状、病歴、入院歴、薬、アレルギー、介助、住環境、希望を自動抽出
- 医師や家族が確認しやすいよう、回答一覧の上部に要点を表示

## v0.3

- 回答内容の感情に合わせて、にゃんこの声かけ文も切り替えるように変更
- 痛み・不安・困惑・安心・重要情報の回答で、より自然な受け止め文を表示
- Claudeの返答が通常寄りでも、重要な回答ではローカル判定の声かけを優先

## v0.2

- 回答内容に応じた簡易感情判定を追加
- 不安・痛み・困惑・安心などに合わせて猫の表情を切り替え
- 表情パターンに `worried` と `relieved` を追加
- Claudeの表情判定とローカル判定を組み合わせ、安定して表情が変わるように調整
